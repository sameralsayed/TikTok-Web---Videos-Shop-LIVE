# 🚀 TikTok Web - Videos, Shop & LIVE

**📌 Description**  
A sleek, fully functional, mobile-first web replica of the official **TikTok** app (com.zhiliaoapp.musically).  
Scroll the For You feed, watch short videos, like & comment, go LIVE, create new clips, discover trending content, and manage your profile — all inside your browser with real video playback and interactions.

**✨ Features**  
- 📱 Vertical swipe For You feed with auto-playing videos  
- ❤️ Like, comment, share & follow directly in feed  
- 🎥 Fully working Create tab (record / upload demo)  
- 🔍 Discover tab with trending grid  
- 📡 LIVE streams + Inbox / Messages  
- 👤 Profile with your videos and stats  
- 🎵 Music stickers & realistic overlays  
- 📲 Bottom navigation exactly like TikTok  
- ♾️ Infinite scroll feel + IntersectionObserver autoplay  
- 🌐 100% responsive on phones, tablets & desktop  

**🏷️ Tags / Keywords**  
`tiktok-web`, `tiktok-clone`, `short-videos-web`, `for-you-feed`, `musically-web`, `live-streaming-demo`, `tiktok-shop`, `video-creator-web`, `viral-videos`, `html5-tiktok`, `browser-tiktok`

**🛠️ How to run**  
1. Download all 4 files into the **same folder**  
2. Open `index.html` in any modern browser (Chrome recommended)  
3. Swipe / scroll the feed and start creating!
